## 实现要点

- **审计日志**：`audit_logs` 表 + `LogHelper`，记录登录、链接/分类增删改，1% 概率触发 90 天自动清理
- **批量操作**：链接批量删除、批量转移分类，按分类维度上移/下移排序（事务保证一致性）
- **导入导出**：原生 JSON + Netscape HTML 书签格式（兼容主流浏览器）；导入走"上传→预览勾选→入库"两阶段，按 URL 幂等查重
- **统计与死链检测**：`click_count` 字段埋点 + cURL 引擎检测链接状态码，VPN 链接单独过滤
- **系统设置**：`settings` 表 + UI 化「私人模式」开关
